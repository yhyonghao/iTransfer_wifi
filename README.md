iTransfer
====
这是一款可以在不同的Android设备之间，Android设备和PC之间传输文件的Android应用。<br>
这是我的第一个开源项目，后续我会继续维护该项目。

#主要技术
-  Android WiFi p2p
-  ftp
-  Material Design
